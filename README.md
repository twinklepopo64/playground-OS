# What is playground-os?
playground-os is planned to be a terminal-focused fake OS that lets you make your own apps, games, whatever you can think of, and run them inside the program!
apps are self-contained in their own folders with their own files and assets!

## How do I make my own apps?
before you start making your own creations, you'll need to install a software called Love2D. It's the framework this project is built on and you'll need it to run your code. (love2d.org)

proper documentation on how to actaully build your apps will come with the first official release, but for now I'm adding as may comments as i can to explain everything while im building this.
